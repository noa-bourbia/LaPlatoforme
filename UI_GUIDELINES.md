# UI Guidelines

## Thèmes
- Client: clair, simple, mobile-first.
- Staff: sombre, gros boutons, action one-tap.
- KDS: très contrasté, large typographie, plein écran.
- Admin: dashboard moderne.

## Statuts (couleurs)
- pending: gris/jaune
- preparing: orange
- ready: vert
- served: bleu/gris
- cancelled: rouge

## Accessibilité
- Contraste AA min.
- Ciblage tactile ≥ 44px.
- Langue FR/EN (i18n).
